# Asia Deals Frontend

Frontend приложение для Asia Deals CRM Integration на Next.js.

## Структура

```
frontend/
├── pages/          # Next.js страницы
├── components/     # React компоненты
├── public/         # Статические файлы
├── utils/          # Утилиты и хелперы
└── styles/         # CSS стили
```

## Запуск

```bash
# Установка зависимостей
npm install

# Разработка
npm run dev

# Сборка
npm run build

# Продакшн
npm start
```

## Основные компоненты

- **Landing Page** - лендинг для привлечения клиентов
- **CRM Dashboard** - панель управления сделками
- **Analytics** - аналитические дашборды
- **Lead Form** - форма приема лидов

## Технологии

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React Query
- Recharts
